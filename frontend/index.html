How it works

      

        
L1 — In-memory
Served from Python dict. Zero disk I/O on repeat calls.

        
L2 — Disk (.md)
One .md per query. Parsed on next boot — no API call needed. Valid Obsidian notes.

        
L3 — Live APIs
OpenReview → arXiv → Semantic Scholar. Written back to L2.

      

      
TTL: config.json → cache.ttl_days (default 7 days).
